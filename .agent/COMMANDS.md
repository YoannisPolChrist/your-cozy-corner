# Workflow Commands

Suggested lightweight commands:

- `/route`
- `/history`
- `/impact`
- `/supervise`
- `/delegate`
- `/accept`
- `/review`
- `/verify`

Rules:

- Keep commands few and obvious.
- Each command should have one clear purpose.
- Commands should reinforce routing, history lookup, delegation, or verification.
- Put long logic in scripts or CI, not in prompt text.
